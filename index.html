Program Keahlian Utama

Teknik Jaringan Komputer dan Telekomunikasi (TJKT)

Jurusan TJKT (sebelumnya TKJ) adalah program keahlian yang membekali peserta didik dengan keterampilan di bidang perakitan komputer, perancangan jaringan LAN/WAN, administrasi server, keamanan siber, hingga instalasi jaringan fiber optik.

Infrastruktur Jaringan

Mempelajari desain, implementasi, dan troubleshooting jaringan LAN, VLAN, WAN menggunakan perangkat router dan switch (Cisco, Mikrotik).

Administrasi Server

Manajemen sistem operasi server (Linux & Windows Server), konfigurasi Web, DNS, DHCP, Mail Server, dan Cloud Computing.

Keamanan Jaringan

Penerapan firewall, proxy, enkripsi data, dan teknik mitigasi serangan siber untuk mengamankan data dan infrastruktur IT.

Sistem Telekomunikasi

Teknologi nirkabel (Wireless), instalasi dan penyambungan (splicing) kabel Fiber Optik, serta dasar jaringan telekomunikasi seluler.

Fasilitas Praktik Standar Industri

Jurusan TJKT SMKN 5 Samarinda dilengkapi dengan laboratorium modern yang dirancang menyerupai environment industri nyata.

  • Laboratorium Jaringan Dasar & Menengah (Mikrotik Academy)
  • Laboratorium Fiber Optik (Splicer & OTDR)
  • Laboratorium Server & Cloud Computing (Rack Server Mount)
  • Perangkat Komputer Core i7 Generasi Terbaru
